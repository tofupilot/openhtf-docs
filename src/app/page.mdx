# OpenHTF

OpenHTF (Open Hardware Testing Framework) is the open-source Python hardware testing framework. {{ className: 'lead' }}

Originally created at **Google**, OpenHTF is designed to streamline hardware testing. It offers out-of-the-box features for structuring tests, recording measurements, logging results and managing external resources with plugs.

OpenHTF is used by hundreds of teams worldwide, from labs to manufacturing floors.
